# Ploop

[**Ploop**](https://ploop.io/) is a set of libraries for facilitating persistence of objects and properties. “Ploop” is somewhat of a backronym, and could stand for any of the following:

* “Persistence Layer for Objects and Properties”
* “Programming Library for Object-Oriented Persistence”

## Modules

Ploop Introspection
: Object discovery and access.
